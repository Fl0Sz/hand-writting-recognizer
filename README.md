# hand-writting-recognizer
YZU 16+2 micro_computer final project
## 專題簡介
本專題實作一套 **手寫數字辨識系統**，整合影像處理技術與深度學習模型，並設計圖形化使用者介面（GUI），可即時展示辨識結果。  
系統支援使用者以滑鼠手寫一整行數字，或上傳包含數字的圖片，系統將自動進行數字切割與辨識，並顯示最終結果。

本專題以 **MNIST 資料集**訓練卷積神經網路（CNN），並應用於實際影像辨識情境，符合 AI 模型整合與系統實作之課程要求。

---

## 系統功能
- 提供手寫畫布，可手寫一整行數字進行辨識
- 支援圖片上傳，辨識圖片中的手寫數字
- 自動進行影像前處理與數字切割
- 即時顯示辨識結果（完整數字串）
- 顯示數字框選結果與二值化除錯影像
- 提供模型準確度評估程式

---

## 系統架構說明
系統主要由五個模組組成，各模組功能如下：
- `gui_app.py`  
  主程式，負責圖形化使用者介面、使用者互動與結果顯示。

- `segment.py`  
  影像前處理與數字切割模組，包含灰階轉換、二值化、形態學處理與連通元件分析，將一行數字切割為單一數字影像。

- `model_infer.py`  
  定義 CNN 模型架構，並載入訓練完成之模型權重進行數字辨識。

- `train_mnist.py`  
  使用 MNIST 資料集訓練 CNN 模型，並將模型權重儲存為 `mnist_cnn.pt`。

- `eval_mnist.py`  
  評估訓練後模型於 MNIST 測試集上的辨識準確率。

## 執行方式
### 1.exe整合包
- 已經把訓練資料全部整合進exe裡
  隨開即用  
  剛開藥等一點時間
  (字寫好看一點成功率很高)
### 2. 本地訓練模型
- cmd裡打 python train_mnist.py
  執行後會產生模型權重檔 mnist_cnn.pt
- cmd打 python gui_app.py
  系統啟動後即可使用手寫畫布或上傳圖片進行辨識
- cmd打 python eval_mnist.py
  會輸出模型於 MNIST 測試集上的整體辨識準確率
---
模型與準確度說明
-  使用模型：卷積神經網路（CNN）
-  訓練資料集：MNIST
-  輸入影像大小：28 × 28（灰階）
-  輸出類別數：10（數字 0–9）
-  模型準確度可透過 eval_mnist.py 進行統計與展示
